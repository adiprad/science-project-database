# REST APIs

Here are the REST APIs supported by the Node-PostgreS application

## USER APIs
POST /api/v1/user
   	- Creates user

DELETE /api/v1/user/{email}
	- Deletes user

GET /api/v1/user/{email}
	- Gets the user info


## Questions Reports APIs
POST /api/v1/question/info
	- Posts a question
